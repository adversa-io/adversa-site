# adversa-site

Marketing Website for Adversa.io

## Overview

This repository contains the static marketing website for Adversa.io, a competitor website monitoring service. The site is integrated with the main application at [https://app.adversa.io](https://app.adversa.io).

## Integration

The website is fully integrated with the main application, providing seamless navigation for users to:
- Sign up for the service
- Log in to existing accounts
- Access specific pricing plans
- View terms, privacy policy, and contact information

For detailed integration documentation, see [INTEGRATION.md](INTEGRATION.md).

## Development

This is a static HTML website that can be hosted on any web server or static hosting service (e.g., GitHub Pages).

### Local Development

Simply open `index.html` in a web browser to view the site locally.

### Deployment

The site is configured for GitHub Pages deployment. Any changes pushed to the main branch will automatically deploy.
